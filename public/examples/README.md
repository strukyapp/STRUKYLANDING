# 🎵 Carpeta de Ejemplos de Audio

Esta carpeta debe contener tus archivos de audio de ejemplo en formato MP3.

## 📁 Archivos requeridos:

- `ejemplo1.mp3` - Pop Romántico (sugerido: "Amor Eterno")
- `ejemplo2.mp3` - Reggaetón (sugerido: "Fuego en la Noche")
- `ejemplo3.mp3` - Trap (sugerido: "Sueños de Libertad")

## 📝 Instrucciones:

1. Prepara 3 ejemplos de canciones producidas (máximo 30-60 segundos cada una)
2. Exporta cada canción en formato MP3 de alta calidad (320kbps recomendado)
3. Coloca los archivos en esta carpeta con los nombres exactos mencionados arriba
4. Los reproductores de audio se actualizarán automáticamente

## 🎨 Personalización:

Para cambiar los nombres, géneros o descripciones, edita el archivo:
`apps/landing/app/page.tsx` en la sección `{/* EXAMPLES SECTION */}`

## ⚠️ Importante:

- **Tamaño máximo recomendado**: 3-5 MB por archivo
- **Formato**: MP3 (mejor compatibilidad)
- **Calidad**: 320kbps o 256kbps
- **Duración**: 30-90 segundos (extractos)

## 💡 Placeholder temporal:

Si aún no tienes los archivos, la página funcionará correctamente.
Los reproductores mostrarán un error solo al intentar reproducir.
